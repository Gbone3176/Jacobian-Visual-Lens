# Limitations

- This is a research visualization utility, not an official model product.
- The included fixture uses synthetic attention and synthetic tokens.
- Real execution depends on Huatuo model internals, support code, and lens compatibility.
- The bundled docs describe the layer16 n50 lens contract, but the binary lens is not included.
- License and citation metadata are pending final review.
- Outputs should not be interpreted as model explanations without additional validation.
